## 説明

**Potago** は、新しい言語を学ぶ方法を変える現代的でゲーミフィケーションされた言語学習プラットフォームです。インタラクティブなフラッシュカード、時間制限付きマッチングゲーム、ビデオコンテンツ、豊富なアチーブメントシステムを組み合わせることで、Potagoは言語学習を楽しく、効果的にします。

最新のAndroidテクノロジー（Kotlin、Jetpack Compose、Material 3）で構築されたPotagoは、すべてのAndroidデバイスで滑らかで美しいユーザーエクスペリエンスを提供します。初心者向けの語彙学習から上級者向けのスキル磨きまで、Potagoはあなたの学習ペースに適応し、インテリジェントなゲーミフィケーションシステムを通じてモチベーションを維持します。

### 対象ユーザー
- すべての熟練度の言語学習者
- 語彙と文法を学ぶ学生
- インタラクティブな学習ツールを求める教育者
- 言語習得に情熱を持つ誰もが

## 機能

### 主要な学習ツール
* **インタラクティブなフラッシュカード** - カスタマイズ可能な語彙セットで語彙を作成、整理、マスターします
* **マッチングゲーム** - 時間制限付きの単語マッチングチャレンジで知識をテストし、最高の時間を競います
* **ライティング練習** - 文型パターンの作成と編集、実例を使用したライティングゲーム
* **ビデオラーニング** - YouTube統合とサブタイトルサポート付きの統合ビデオライブラリ
* **多言語対応** - 柔軟な言語ペアの設定で、任意の言語間で学習できます

### ゲーミフィケーション & エンゲージメント
* **経験値ポイントシステム** - 完了したすべての学習活動でXPを獲得
* **デイリーストリーク** - 一貫した学習習慣を維持し、ストリークを保護します
* **アチーブメントバッジ** - 学習マイルストーンでスペシャルアチーブメントをアンロック
* **リーダーボード & ランキング** - 他の学習者と競争し、進捗を追跡
* **仮想報酬** - ダイヤモンドを集めて、プレミアムブーストと機能をアンロック
* **マスコットコンパニオン** - Potagoを紹介、あなたと一緒に成長する学習コンパニオン

### コンテンツ管理
* **個人用語セット** - 独自の語彙コレクションを構築するか、コミュニティセットを閲覧
* **ビデオライブラリ** - キュレーションされた教育ビデオへのアクセスまたは自分の学習リソースをアップロード
* **学習履歴** - 最近のアクティビティと時間経過による進捗を追跡
* **ソーシャル共有** - 友人やコミュニティと語彙セットを共有

### ユーザー機能
* **セキュア認証** - Firebase対応の認証によるアカウントセキュリティ
* **カスタマイズ可能なプロフィール** - アバターをアップロードして学習スペースをカスタマイズ
* **柔軟な設定** - 通知、言語設定、学習目標を制御
* **クラウド同期** - すべてのデバイス間で進捗が同期されます
* **アプリ内ショップ** - ストリーク保護やエクスペリエンスブーストなどの便利なアイテムを購入

## インストールと更新

### 1. GitHubリリースからAPKをダウンロード（推奨）
[GitHubリリース](https://github.com/duanhap/PotagoApp/releases)ページから最新のAPKを直接ダウンロードしてください。

```bash
# APK署名を検証する（オプション、推奨）
# SHA-256: [YOUR_SHA_256_HASH]
```

### 2. ソースからビルド
リポジトリのクローンとAndroid Studioまたはgradleを使用したビルド:

```bash
git clone https://github.com/duanhap/PotagoApp.git
cd PotagoApp
./gradlew assembleDebug  # デバッグAPKをビルド
./gradlew assembleRelease  # リリースAPKをビルド
```

ビルドされたAPKは `app/build/outputs/apk/` にあります。

### 3. F-Droid（将来）
F-Droidサポートに取り組んでいます。お待ちください！

### インストール手順
1. Android設定 > セキュリティで「不明なソース」を有効にします
2. APKファイルをダウンロード
3. APKを開いて「インストール」をタップ
4. 必要なパーミッション（インターネット、通知など）を許可
5. Potagoを起動してアカウントを作成

### 更新
今後のアップデートはGitHubリリースページで利用可能です。新しいAPKをダウンロードしてインストールするだけで更新できます。

> [!tip]
> 更新前に進捗をバックアップ: 設定 > バックアップと復元 > データベースをエクスポート

> [!warning]
> **THIS APP IS IN ACTIVE DEVELOPMENT.** バグや不完全な機能に遭遇する可能性があります。問題が見つかった場合は、[GitHubリポジトリ](https://github.com/duanhap/PotagoApp/issues)に報告してください。

## 使用方法

### はじめに
1. **アカウントを作成** - メールまたは既存の認証情報でサインアップ
2. **学習目標を設定** - ターゲット言語と学習ペースを選択
3. **最初の語彙セットを作成** - マスターしたい語彙を追加
4. **学習を開始** - フラッシュカードから始めて、マッチングゲームを試す
5. **進捗を追跡** - ストリーク、XP、アチーブメントを監視

### 学習ワークフロー
- **フラッシュカード学習** - 自分のペースでカードを学習
- **マッチングゲーム** - 時間制限付きのチャレンジで知識をテスト
- **ライティング練習** - 構造化された演習で文法を完璧に
- **ビデオを視聴** - キュレーションされたビデオコンテンツで学習を補強

### 成功のコツ
- デイリーストリークを維持して一貫した進捗を達成
- 単調さを避けるために異なる学習モードを混ぜる
- 難しい単語をより頻繁にレビュー
- 進むにつれてショップでストリーク保護を購入
- 友人とセットを共有して一緒に学習

## プライバシー

Potagoはあなたのプライバシーを尊重します。知っておくべきことは次のとおりです：

**データ収集**
- ユーザーアカウントデータ（メール、プロフィール情報）認証用
- 学習進捗と統計情報（パーソナライズされた推奨を提供）
- クラッシュレポート（アプリの安定性向上、オプション）

**収集しないデータ**
- Potago外の個人用閲覧履歴
- デバイス識別子または広告データ
- ロケーションデータやカメラへのアクセス
- 連絡先リストまたは個人ファイル

**データストレージ**
- アカウントデータは暗号化されてサーバーに保存されます
- 学習進捗はデバイスに保存され、アカウントと同期されます
- 設定からいつでもデータをエクスポートまたは削除できます

詳細については、[プライバシーポリシー](#)を参照してください。
